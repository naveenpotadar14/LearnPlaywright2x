# Chapter 07: If-Else Statements

This chapter covers JavaScript conditional logic using `if-else` statements, including nested conditions and practical real-world examples.

## Files Overview

| File | Description |
|------|-------------|
| `48_If_Else.js` | Basic if-else conditional structure and syntax |

| `50_real_if_else.js` | Real-world practical examples of if-else logic |
| `51_api_if_else.js` | Conditional logic with API-like scenarios |

## Key Concepts

- **Basic Conditionals**: if, else, else-if statements
- **Nested Conditions**: Multiple levels of conditional logic
- **Logical Operators**: AND (&&), OR (||), NOT (!)
- **Comparison Operators**: ==, ===, !=, !==, <, >, <=, >=
- **Practical Applications**: Grade systems, leap year detection, number validation

## Running Examples

Run any example file with Node.js:

```bash
node 48_If_Else.js
node 57_grade_cal.js
node 58_leap_year.js
```

## Learning Path

1. Start with `48_If_Else.js` for basic syntax
2. Progress to `49_if_elseif_else.js` for multiple conditions
3. Review `50_real_if_else.js` and `53_If_Else_Real.js` for practical patterns
4. Practice with `56_IQ_ODD_EVEN.js` for logic verification
5. Apply knowledge to `57_grade_cal.js` and `58_leap_year.js` for complex scenarios
6. Challenge yourself with `52_IQ_If_Else.js`, `54_IQ.js` for interview questions
