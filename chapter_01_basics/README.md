# Chapter 01 Basics

This folder contains four introductory JavaScript files used for early Playwright and Node.js practice.

## Files

### `01_Basics.js`
A minimal startup script that prints a short greeting to the console and defines a single numeric variable. Use this file to verify that Node.js is running and that you can execute simple JavaScript files.

### `02_Js.js`
A small JavaScript example demonstrating variable assignment, console output, loops, and function declaration. This file also contains a loop that prints numbers from 0 to 99,999 and calls a helper `print()` function.

### `03_VerifySetup.js`
A setup verification script that prints environment information from Node.js. It logs the operating system platform, CPU architecture, and the installed Node.js version.

### `04_HotCode.js`
A performance-style example that prints a greeting, defines an `add` function, runs the function inside a loop for 10,000 iterations, and shows the final computed result.

## How to run

From the repository root, run any script with Node.js:

```bash
node chapter_01_basics/01_Basics.js
node chapter_01_basics/02_Js.js
node chapter_01_basics/03_VerifySetup.js
node chapter_01_basics/04_HotCode.js
```

These scripts are useful for verifying your local environment and learning basic JavaScript execution flow before moving on to Playwright-specific examples.
