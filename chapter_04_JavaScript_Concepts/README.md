## How to run the examples
Use Node.js (v12+) to run examples locally. From the workspace root run:

```bash
node "chapter_04_JavaScript_Concepts/12_let_explained.js"
```

Or open the file in VS Code and run snippets in the integrated terminal or the editor run buttons.
