# VS Code Keyboard Shortcuts - Windows

## File Operations
| Action | Shortcut |
|--------|----------|
| New File | `Ctrl + N` |
| Open File | `Ctrl + O` |
| Open Folder | `Ctrl + K Ctrl + O` |
| Save | `Ctrl + S` |
| Save All | `Ctrl + Shift + S` |
| Close Tab | `Ctrl + W` |
| Close All Tabs | `Ctrl + K Ctrl + W` |
| Reopen Closed Tab | `Ctrl + Shift + T` |

## Editing
| Action | Shortcut |
|--------|----------|
| Cut Line | `Ctrl + X` |
| Copy Line | `Ctrl + C` |
| Paste | `Ctrl + V` |
| Undo | `Ctrl + Z` |
| Redo | `Ctrl + Shift + Z` |
| Select All | `Ctrl + A` |
| Delete Line | `Ctrl + Shift + K` |
| Duplicate Line | `Ctrl + Shift + D` |
| Move Line Up | `Alt + Up` |
| Move Line Down | `Alt + Down` |
| Comment Line | `Ctrl + /` |
| Block Comment | `Shift + Alt + A` |
| Go to Line | `Ctrl + G` |
| Find | `Ctrl + F` |
| Replace | `Ctrl + H` |
| Multi-Cursor | `Ctrl + D` |
| Add Cursor to Next Match | `Ctrl + D` (repeat) |
| Add Cursor to All Matches | `Ctrl + Shift + L` |
| Create Multiple Cursors | `Ctrl + Alt + Down/Up` |
