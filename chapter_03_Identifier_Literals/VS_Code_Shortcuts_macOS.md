# VS Code Keyboard Shortcuts - macOS

## File Operations
| Action | Shortcut |
|--------|----------|
| New File | `Cmd + N` |
| Open File | `Cmd + O` |
| Open Folder | `Cmd + K Cmd + O` |
| Save | `Cmd + S` |
| Save All | `Cmd + Shift + S` |
| Close Tab | `Cmd + W` |
| Close All Tabs | `Cmd + K Cmd + W` |
| Reopen Closed Tab | `Cmd + Shift + T` |

## Editing
| Action | Shortcut |
|--------|----------|
| Cut Line | `Cmd + X` |
| Copy Line | `Cmd + C` |
| Paste | `Cmd + V` |
| Undo | `Cmd + Z` |
| Redo | `Cmd + Shift + Z` |
| Select All | `Cmd + A` |
| Delete Line | `Cmd + Shift + K` |
| Duplicate Line | `Cmd + Shift + D` |
| Move Line Up | `Option + Up` |
| Move Line Down | `Option + Down` |
| Comment Line | `Cmd + /` |
| Block Comment | `Shift + Option + A` |
| Go to Line | `Cmd + G` |
| Find | `Cmd + F` |
| Replace | `Cmd + H` |
| Multi-Cursor | `Cmd + D` |
| Add Cursor to Next Match | `Cmd + D` (repeat) |
| Add Cursor to All Matches | `Cmd + Shift + L` |
| Create Multiple Cursors | `Cmd + Option + Up/Down` |

## Navigation
| Action | Shortcut |
|--------|----------|
| Quick Open File | `Cmd + P` |
| Go to Symbol | `Cmd + Shift + O` |
| Go to Line | `Cmd + G` |
| Peek Definition | `Cmd + Option + Shift + Right Arrow` |
| Go to Definition | `F12` or `Cmd + Right Click` |
| Go Back | `Cmd + Option + Left` |
| Go Forward | `Cmd + Option + Right` |
| Next Error | `F8` |
| Previous Error | `Shift + F8` |

## View & Layout
| Action | Shortcut |
|--------|----------|
| Toggle Sidebar | `Cmd + B` |
| Toggle Panel | `Cmd + J` |
| Toggle Terminal | `Cmd + ~` |
| Focus Explorer | `Cmd + Shift + E` |
| Focus Search | `Cmd + Shift + F` |
| Focus Debug | `Cmd + Shift + D` |
| Focus Extensions | `Cmd + Shift + X` |
| Zen Mode | `Cmd + K Cmd + Z` |
| Toggle Word Wrap | `Option + Z` |
| Zoom In | `Cmd + +` |
| Zoom Out | `Cmd + -` |
| Reset Zoom | `Cmd + 0` |
| Split Editor | `Cmd + \` |
| Switch to Next Editor | `Cmd + Tab` |
| Switch to Previous Editor | `Cmd + Shift + Tab` |

## Refactoring
| Action | Shortcut |
|--------|----------|
| Rename Symbol | `F2` |
| Format Document | `Shift + Option + F` |
| Format Selection | `Cmd + K Cmd + F` |
| Quick Fix | `Cmd + .` |
| Show References | `Shift + F12` |

## Debugging
| Action | Shortcut |
|--------|----------|
| Start Debug | `F5` |
| Continue | `F5` |
| Pause | `F6` |
| Step Over | `F10` |
| Step Into | `F11` |
| Step Out | `Shift + F11` |
| Toggle Breakpoint | `F9` |

## Terminal
| Action | Shortcut |
|--------|----------|
| New Terminal | `Cmd + Shift + ~` |
| Focus Terminal | `Cmd + ~` |
| Kill Terminal | `Cmd + Shift + ~` (while focused) |
