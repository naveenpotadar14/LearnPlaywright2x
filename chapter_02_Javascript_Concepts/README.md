# Chapter 02 JavaScript Concepts

This folder contains JavaScript concept examples used for learning and practicing Playwright automation.

## Files

### `05_JS.js`
A simple JavaScript example demonstrating variable declaration using `var` and variable reassignment. The script declares a variable, logs its initial value, reassigns it to a new value, and logs the updated value.

## How to run

From the repository root, run the script with Node.js:

```bash
node chapter_02_Javascript_Concepts/05_JS.js
```

This script is useful for understanding basic variable manipulation and scoping in JavaScript before applying these concepts to Playwright test automation.
